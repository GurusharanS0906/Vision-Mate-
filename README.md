Vision Mate is an IoT-based assistive device designed to help visually impaired individuals navigate safely.

## Project Overview
The smart walking stick uses an ESP32 microcontroller and ultrasonic sensors to detect nearby obstacles and provide alerts to the user in real time.

## Hardware Components
- ESP32 Microcontroller
- Ultrasonic Sensor
- Power Supply
- Buzzer / Alert Module

## Features
- Obstacle detection using ultrasonic sensors
- Real-time distance-based alerts
- Compact and portable design

## Future Enhancements
- Live location tracking using GPS
- Mobile app integration for caretakers
- Fall detection and orientation monitoring
- Emergency alert system

- ESP32
- IoT & Embedded Systems
- Sensor Integration
- C / Embedded Programming
